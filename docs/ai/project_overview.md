# Project Overview

Short summary for humans and AI.

## What it is
Smart Campus Academic Management System: a Flask + MySQL app for student enrollment, attendance, grades, and admin reporting.

## Stack
MySQL 8, Python 3.11, Flask 3.x, mysql-connector-python, Jinja2, vanilla CSS/JS, Windows 11.

## Roles
| Role | Main actions |
|---|---|
| Student | enroll, view attendance, grades, transcript |
| Faculty | view courses, roster, mark attendance, enter grades |
| Admin | manage students, faculty, courses, reports |

## Dev login
`student / 1234`, `faculty / 1234`, `admin / 1234`.

## Core scope
- Server-rendered Flask app with session auth.
- MySQL schema with procedures, views, and triggers.
- Modern Clean Slate UI with light/dark mode.

## Status
- Backend, templates, CSS, and JS are in place.
- Real DB wiring, seed data, and production auth still need completion.
